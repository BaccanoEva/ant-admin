# ant-admin
此项目基于nelsonkuang/ant-admin(https://github.com/nelsonkuang/ant-admin)

## 如何使用?

此项目使用`Create React App`初始化的   
npm下可用的命令   

``` 
npm start   
npm test   
npm run build   
npm run eject   

```

yarn下可用的命令   

``` 
yarn start   
yarn test   
yarn build   
yarn eject   

```
